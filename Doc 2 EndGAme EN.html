Installation

Step-by-step setup

This section completes the quick setup with the missing details that actually matter: the correct folder, the helper tools, the MIDI checks, and the key things to verify if something fails.

Folder and expected content

  • Typical path: C:\Users\YourName\Documents\Image-Line\FL Studio\Settings\Hardware
  • Depending on your setup, it may also be inside the FL Studio data folder.
  • The folder must contain device_Fire.py and the fire_modules subfolder.
  • If FL Studio cannot see the script, the first thing to check is simply the folder location.

Tools and purpose

  • AkaiFireWatcher.exe: required for the project, and should be launched before FL Studio.
  • FLControlEditor.exe: used to configure custom CTRL FL shortcuts.
  • color_picker_to_0x.html: helper page for choosing pad color values.
  • You can create desktop shortcuts for these tools to launch them faster.
Step Action Details
1 Copy the Akai Fire EndGAme folder The folder must be placed inside the FL Studio Hardware scripts folder.
2 Create desktop shortcuts Inside the main folder and fire_modules, create shortcuts for AkaiFireWatcher.exe, FLControlEditor.exe, and color_picker_to_0x.html.
3 Launch AkaiFireWatcher.exe The watcher must already be active before opening FL Studio.
4 Power on the Akai Fire Wait for full initialization before moving on.
5 Launch FL Studio Then open MIDI settings to complete the setup.
6 Select the MIDI script Choose Akai Fire EndGAme as the Fire script.
7 Set the MIDI ports Use exactly the same port number for input and output.
8 Verify operation Test STEP, NOTE, CTRL FL, and PERFORM to confirm everything responds correctly.
Tip

If the script does not react correctly, check the install folder first, then the selected MIDI script, then the MIDI port, then the launch order: watcher, Fire, FL Studio.

STEP Mode

The sequencer reinvented

STEP mode gives you four different ways to work. The first one keeps the original Fire spirit, while the other three switch to a true vertical sequencing concept.

Sub-mode Shortcut Concept
STEP 1 - Classic SHIFT + STEP A redesigned version of the original mode. It uses a 1 x 64 step layout instead of 4 x 16. You control one selected channel rack lane over 64 visible steps.
STEP 2 - Vertical MONO STEP The 64-pad matrix shows 4 steps of the sequencer across 16 note heights. One note per step.
STEP 3 - Vertical POLY ALT + STEP Same vertical concept as MONO, but you can freely add multiple notes on the same step.
STEP 4 - Vertical POLY CHORDS ALT + SHIFT + STEP Same vertical grid, but the chord selected with BROWSER is automatically added when you place a note.

Navigation inside the grid

  • SELECT moves the visible window 4 steps at a time.
  • PATTERN - and PATTERN + move one step at a time.
  • The first knob shifts the visible pitch range.
  • With multiple synced Fire units, you can display 8 steps or 16 steps at once.

EDIT mode

  • Press STEP again and the button blinks to enter EDIT mode.
  • You can select one or more notes in the grid.
  • Knob 1 controls pitch.
  • Knob 2 controls velocity.
  • Knob 3 controls note length.
  • Knob 4 controls timing offset.
  • MUTE 1 to MUTE 4 mute the 4 visible steps.
Command Action
BROWSER Opens the chord selection mode, useful in STEP 4 and NOTE chord mode.
SELECT while already in vertical STEP Syncs the internal grid back to FL Studio.
ALT + press SELECT Exports the sequence as both MIDI and JSON.
SHIFT + press SELECT Reloads a saved sequence from disk.
Selector Cycles quickly through STEP 1, STEP 2, STEP 3 and STEP 4.
Multi-Fire

Multi-device sync is mainly intended for the vertical STEP modes. The idea is to split the visible steps across several Fire units to show a larger sequence area.

NOTE Mode

The keyboard with automatic chords

NOTE mode turns the pad matrix into a chromatic keyboard with four visible octaves. It can also play full chords directly.

Sub-mode Shortcut Concept
Classic NOTE NOTE A chromatic keyboard with 4 visible octaves. White and black notes are arranged in sequence on each row.
Chord NOTE Press SELECT The keyboard plays the selected chord directly instead of a single note.

Useful commands

  • PATTERN - and PATTERN + change the 4 visible octaves.
  • BROWSER opens chord selection.
  • SELECT changes the selected instrument or channel.
  • Press SELECT again to enable or disable chord mode.

Chord selection

  • BROWSER enters chord selection mode.
  • Turning SELECT scrolls through chord types.
  • The chord name scrolls directly on the pad matrix.
  • BROWSER closes the mode and returns to the previous one.
CTRL FL Mode

The Fire becomes a dedicated FL Studio controller

Each pad becomes a shortcut or direct action inside FL Studio. The goal is to control the mixer, channel rack, playlist and piano roll quickly without relying on the mouse.

Area Role
Row 1 Select mixer tracks or channel rack channels. SHIFT + pad mutes the target.
Row 2 Pattern selection.
Row 3 Playlist shortcuts, plus fixed keys such as Escape, Up and Enter.
Row 4 Piano roll shortcuts, plus Left, Down and Right arrows.
Command Function
MUTE 1 Switches the first row to Mixer view, then toggles between pages 1-16 and 17-32.
MUTE 2 Switches the first row to Channel Rack view, then toggles between pages 1-16 and 17-32.
MUTE 3 Toggles playlist shortcuts between page A and page B.
MUTE 4 Toggles piano roll shortcuts between page A and page B.
Double press on a Channel Rack pad Opens the channel editor.
Touch KNOB 1, 2, 3, 4 Opens Channel Rack, Piano Roll, Mixer and Playlist respectively.

Customization

  • The shortcut pads are fully configurable.
  • You can define the key, modifiers, label and color for each pad.
  • FLControlEditor.exe is made for configuring everything cleanly.

What it brings

  • Direct access to playlist tools.
  • Direct access to piano roll tools.
  • Fast navigation through FL Studio windows.
  • A much more immediate physical workflow than the original Fire behavior.
PERFORM Mode

The live sync engine

This mode does not use FL Studio’s native performance mode. It acts as its own live system, able to mute, relaunch or switch patterns in sync.

Selection Controlled source
MUTE 1 Mixer
MUTE 2 Channel Rack
MUTE 3 Playlist
MUTE 4 Patterns
Situation Behavior
FL Studio is stopped The action happens immediately.
FL Studio is playing The pad enters a pending state, then the action is applied at the next sync boundary.
SHIFT + PAD Forces the action immediately without waiting for sync.
Patterns mode The current pattern length is detected automatically so switching happens at the correct moment.

Useful commands

  • SELECT chooses the sync length: 1, 2, 4 or 8 bars.
  • Pressing SELECT can also rotate through Mixer, Channel Rack, Playlist and Patterns.
  • SHIFT + GRID L/R changes page by blocks of 64 elements.

Typical live use

  • Queue several mute changes in advance.
  • Launch a variation on the next measure.
  • Switch patterns automatically at the real end of the current pattern.
  • Create live interventions on tracks, channels or patterns.
Limitations

What to know about vertical STEP mode

The vertical sequencer cannot be displayed inside FL Studio exactly as it appears on the Fire. This is a limitation of the piano roll API.

Main limitation

In FL Studio, notes remain visible like simple steps. In vertical POLY or POLY CHORD STEP modes, several notes may be played on the Fire, but FL often only shows the lowest note of the step.

Result

  • On the Fire, you still see and hear the real chords.
  • Inside FL, you do not always see every note that is actually being played.
  • The piano roll often behaves as if it were only showing a simple channel rack step.

Built-in solution

  • ALT + press SELECT exports the sequence as a MIDI file.
  • SHIFT + press SELECT reloads an existing sequence.
  • This lets you recover the real sequence in FL Studio and edit it afterward.
